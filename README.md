# Ultra Weather X

Eine moderne Premium-Wetter-App mit HTML, CSS und JavaScript.

## Features

- Städte suchen
- Aktuelles Wetter anzeigen
- Temperatur
- Wetter-Icons
- Windgeschwindigkeit
- Luftfeuchtigkeit
- Glassmorphism-Design
- Dynamische Hintergründe
- Quick Buttons
- Live Uhr
